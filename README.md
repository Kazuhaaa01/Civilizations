# Civilizations

## Descripció del projecte

**Civilizations** és un projecte de desenvolupament de programari creat com a part del treball de curs. L'objectiu és dissenyar i implementar un sistema modular organitzat en cinc mòduls (M1–M5), cadascun responsable d'una funcionalitat específica del projecte.

## Estructura del projecte

```
Civilizations/
├── M1/   # Mòdul 1
├── M2/   # Mòdul 2
├── M3/   # Mòdul 3
├── M4/   # Mòdul 4
├── M5/   # Mòdul 5
└── README.md
```

## Requisits previs

- [Git](https://git-scm.com/) instal·lat al sistema.
- Les eines i dependències específiques de cada mòdul es detallen dins de la seva carpeta corresponent.

## Instal·lació

1. Clona el repositori:

   ```bash
   git clone https://github.com/Kazuhaaa01/Civilizations.git
   cd Civilizations
   ```

2. Accedeix al mòdul que vulguis treballar:

   ```bash
   cd M1   # o M2, M3, M4, M5
   ```

3. Segueix les instruccions d'instal·lació específiques del mòdul (vegeu el `README.md` de cada carpeta).

## Execució

Cada mòdul conté les seves pròpies instruccions d'execució al seu `README.md`. En general:

```bash
cd <Mòdul>
# Segueix les instruccions del README.md del mòdul
```

## Branques del repositori

| Branca | Descripció |
|--------|------------|
| `main` | Branca principal amb el codi en producció |
| `preproduction` | Branca de preproducció on es fan els *merges* dels membres de l'equip |
| `<alumne>` | Branca individual de treball de cada membre de l'equip |

## Control de versions

- Tots els canvis han d'anar acompanyats d'un missatge de *commit* descriptiu.
- Cada membre de l'equip treballa a la seva pròpia branca i consolida els canvis a `preproduction`.
- El lliurament final es farà en format *release* a GitHub.

## Llicència

Aquest projecte és de caire acadèmic.
